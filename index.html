React

React là một thư viện JavaScript để xây dựng giao diện người dùng.
React được sử dụng để xây dựng các ứng dụng một trang.
React cho phép chúng ta tạo các thành phần UI có thể tái sử dụng.

NodeJS

Node.js là một môi trường máy chủ mã nguồn mở.
Node.js cho phép bạn chạy JavaScript trên máy chủ.

PHP

PHP là một ngôn ngữ kịch bản máy chủ và là một công cụ mạnh mẽ để tạo các trang Web động và tương tác.
PHP là một giải pháp thay thế được sử dụng rộng rãi, miễn phí và hiệu quả cho các đối thủ cạnh tranh như ASP của Microsoft.

MySQL

MySQL là một hệ quản trị cơ sở dữ liệu quan hệ (RDBMS) được sử dụng rộng rãi.
MySQL là mã nguồn mở miễn phí.
MySQL lý tưởng cho cả các ứng dụng nhỏ và lớn.
